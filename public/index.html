◆ DEDLIT Studio

Tu frontend local y privado de IA. Selecciona un proveedor y un modelo en la barra lateral.

  • Local: Ollama y LM Studio se detectan automáticamente — nada sale de tu equipo.
  • 🧲 Modelos: busca en Hugging Face y descarga GGUF para LM Studio u Ollama, con semáforo 🟢🟡🔴 según tu RAM y GPU.
  • Nube: Claude, OpenAI, Gemini, Grok, DeepSeek, Qwen, Kimi y GLM con tu propia API key (⚙ Ajustes).
  • ¿Sin API key? Pulsa 🎁 APIs gratis: OpenRouter, Gemini, Groq, Cerebras, GitHub Models y más ofrecen nivel gratuito.
  • Modo agente: el modelo puede leer/buscar/escribir archivos, usar la terminal, leer páginas web y abrir tu navegador — con tu aprobación.
  • Conectores MCP: añade en ⚙ Ajustes los mismos conectores de Claude Desktop (control de Chrome con Playwright, GitHub, bases de datos…).
  • Multimedia: adjunta imágenes, audio o vídeo con 📎 (según el modelo), y genera imágenes 🖼️, voz 🔊 o transcripciones 📝 con el selector de modo.
  • VS Code: apunta Continue/Cline al gateway http://127.0.0.1:8642/v1.
  • Atajos: Ctrl+K buscar · Ctrl+Shift+O nuevo chat · Ctrl+, ajustes · Alt+A modo agente · Esc cerrar.

📚 Documentos (RAG local)

Indexa una carpeta (texto, código, Markdown, PDF y DOCX) para que el modelo responda usando tus documentos como contexto, citando el archivo. Los embeddings se calculan con el proveedor que elijas — recomendado local (LM Studio u Ollama con un modelo de embeddings como nomic-embed-text): así nada sale de tu equipo. 🧲 Buscador de modelos (Hugging Face)

Detectando tu hardware…

Resultados en formato GGUF (el que usan LM Studio y Ollama), ordenados por descargas. El semáforo se calcula con tu RAM y VRAM: 🟢 cabe entero en GPU · 🟡 parcial GPU+CPU o solo CPU · 🔴 no cabe. La búsqueda consulta huggingface.co solo cuando tú la lanzas.

🎁 APIs de IA gratuitas

¿No tienes API keys? Estos servicios ofrecen nivel gratuito — creas una cuenta, generas la key y la pegas en ⚙ Ajustes. Los límites cambian a menudo; lista completa y actualizada en cheahjs/free-llm-api-resources.

⚠ Ten en cuenta que algunos niveles gratuitos pueden usar tus conversaciones para entrenar modelos — revisa las condiciones de cada servicio si la privacidad es crítica. Para privacidad total, usa Ollama o LM Studio en local.

Idioma de la interfaz

Workspace del agente

Carpeta raíz donde el agente trabaja por defecto.

Carpeta de modelos de LM Studio

Donde el buscador 🧲 descarga los GGUF para LM Studio. Vacío = la carpeta estándar.

Stable Diffusion local

URL de Automatic1111 / SD.Next / Forge (arráncalo con --api). Si está activo, la generación de imágenes lo usa automáticamente: gratis y 100% privada.

ComfyUI local

Backend alternativo de imágenes. Prioridad automática: SD WebUI → ComfyUI → nube.

Voz local (opcional)

Servidores locales OpenAI-compatibles para 📝 transcribir y 🔊 generar voz sin nube. Si se rellenan, tienen prioridad sobre el proveedor seleccionado.

Instrucciones personalizadas

Se añaden como instrucciones de sistema a todas tus conversaciones (chat y agente).

Aprobaciones automáticas del agente

Lo que no esté marcado te pedirá aprobación en el chat antes de ejecutarse.

Conectores (MCP)

DEDLIT es cliente de Model Context Protocol, el estándar de conectores de Claude Desktop: navegador, Canva, GitHub, bases de datos… Mismo formato de configuración. Ejemplos (requieren Node en el PATH); los servidores remotos como Canva abren el navegador la primera vez para autorizar tu cuenta:

{
  "navegador": { "command": "npx", "args": ["-y", "@playwright/mcp@latest"] },
  "canva": { "command": "npx", "args": ["-y", "mcp-remote", "https://mcp.canva.com/mcp"] }
}

API keys y endpoints

Las keys se guardan cifradas (AES-256-GCM) en tu carpeta de usuario y nunca salen de tu equipo salvo hacia el proveedor correspondiente. Deja el campo vacío para no cambiar; escribe - para borrar una key.
