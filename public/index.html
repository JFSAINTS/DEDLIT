◆ DEDLIT Studio

Tu frontend local y privado de IA. Selecciona un proveedor y un modelo en la barra lateral.

  • Local: Ollama y LM Studio se detectan automáticamente — nada sale de tu equipo.
  • Nube: Claude, OpenAI, Gemini, Grok, DeepSeek, Qwen, Kimi y GLM con tu propia API key (⚙ Ajustes).
  • Modo agente: el modelo puede leer/escribir archivos y ejecutar comandos con tu aprobación.
  • Multimedia: adjunta imágenes, audio o vídeo con 📎 (según el modelo), y genera imágenes 🖼️, voz 🔊 o transcripciones 📝 con el selector de modo.
  • VS Code: apunta Continue/Cline al gateway http://127.0.0.1:8642/v1.

⚙ Configuración

Workspace del agente

Carpeta raíz donde el agente trabaja por defecto.

Aprobaciones automáticas del agente

Lo que no esté marcado te pedirá aprobación en el chat antes de ejecutarse.

API keys y endpoints

Las keys se guardan cifradas (AES-256-GCM) en tu carpeta de usuario y nunca salen de tu equipo salvo hacia el proveedor correspondiente. Deja el campo vacío para no cambiar; escribe - para borrar una key.
